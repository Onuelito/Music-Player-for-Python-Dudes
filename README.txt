In order to run this project you will need the following:

- Python 3
- Pygame
- Mutagen

Python 3 can be installed in this link : https://www.python.org/
Pygame instructions to install is in this link : https://www.pygame.org/wiki/GettingStarted
Mutagen instruction to install is in this link : https://mutagen.readthedocs.io/en/latest/
